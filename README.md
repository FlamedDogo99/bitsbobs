## Bits & Bobs
A Fabric 1.21.10 server-side mod that adds various bits and bops through polymer

### Current Bits and Bobs
- Netherite Stairs
  - Walkable! 
- Seats (of all the wool colors)
  - Sittable and Bounceable!
- Nether Brick Fence Gate
  - Flappable! 
- Dirt Slab
- Suspicious Flint and Steel
- Blood Fire
- Saddle on a Stick
  - Ride _any_ entity!
- Rose & Blue Rose
